# Sobre o projeto
Um server gRPC em Go.
Projeto criado segundo as aulas [Como utilizar o gRPC com GO!](https://www.youtube.com/watch?v=LuS59XHdKG8&list=WL&index=17)

# Como utilizar
1 - Rode o server
```
go run cmd/server/main.go
```
1 - Rode o client
```
go run cmd/client/main.go
```
